Twitch API とサービス

Twitch API は公開されています。デベロッパの方向け…程には書くスキルがないのですが、紹介いたします。 https://github.com/justintv/Twitch-API

Twitch API

もしかして、Twitchのことを書くのは初めてでしょうか？
というわけで、上のリンクでTwitchのAPIは出ています。これを用いて開発が可能です。

Twitch Alerts や Night Dev といった、生放送の体験を拡張する機能から、#TwitchPlaysPokemon で一度Twitterを沸かせた企画まで、色々出来ます。
