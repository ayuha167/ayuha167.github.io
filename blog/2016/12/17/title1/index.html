iOSからTwitchへ配信

タイトルのとおりです。

こちらの投稿：http://ayuha167.github.io/blog/2016/12/17/

が元です。

2021年1月、Twitch アプリから配信可能になりました。 詳しくはこちら

2017年7月現在

Twitch アプリ ver5.0 内から、直接配信が可能です。是非ご利用下さい。

#iOS デバイス

  1. App Store から Live Air Solo をダウンロードしてください。 https://itunes.apple.com/us/app/live-air-solo-stream-live/id1051147032

  1. アプリ、起動。

  2. カメラ・マイク・写真・位置情報 へのアクセスを許可するように訊かれたら「OK」を押します。

  3. Destination に Twitch を選びます。

  4. ユーザー名・パスワードを入力して Twitch のアカウントへログイン。 2段階認証を用いている方は、Authy/SMS からのコードを、パスワードの文字列につづいてパスワードボックスに直接打ち込んで下さい。

  5. お近くのサーバーを選んで下さい。（皆様はたぶん Asia : Tokyo, Japan です）

［ココから配信画面の設定です］

  1. 【カメラ】 画面左端を真ん中へスワイプしてください。Sources バーが表示されます。（ここから、フロント/バック カメラを配信映像として選べます。） Add New Source をクリックしてください。次に iOS Camera を選択します。そして、サムネイルを選択して、iOSカメラに切り替えます。

  1. 【オーバーレイ】 オーバーレイを変更・削除する場合は、画面右端を真ん中へスワイプします。オーバーレイメニューバーの中にあるサムネイルをクリックすることで、オーバーレイのパーツを追加/削除 することが出来ます。長押しすれば編集が出来ます。

Add New Overlay を押せば、自分で作成した画像をオーバーレイに出来ます。

  1. これで基本は準備完了です！ 画面上端を下へスワイプすると、Go Live ボタンが現れます。こちらを押すとライブストリーミング開始です！

Live:Air Solo にログインできない場合

（2017年2月追記）「Live:Air Solo に入ろうとしても、Twitch のアカウント認証が出来ない！エラーが起きて弾かれる…」こういった問題が一定期間に一度現れます。その場合は、Twitch ではなく “Custom RTMP” を選択して下さい。

そして設定は

  • URL : rtmp://live-tyo.twitch.tv/app/
  • Stream : （live_ から始まる配信キー）

を入力して、それ以外は放置でお願いします。

URLにあるアドレスは、Twitch のIngest Server です。こちらから世界のモノを一覧できます。https://bashtech.net/twitch/ingest.php

また、配信キーは、
https://www.twitch.tv/(自分のアカウント名)/dashboard/streamkey
から取得できます。

お手数おかけしますが、この形式でお願いします。

#おまけ :

■ 解像度の変更 画面上端から下へスワイプして、Live バーを表示して下さい。左の方にある青いギアマークを押しましょう。

OUTPUT の下の Encoder Settings をクリックし、Manual を選択します。良いインターネット接続がある場合は、 720p, 3500kbps, 30fps, 2s Keyframe Intervals, Adaptive Bitrate ON, Bitrate Limiting OFF をオススメします。

■Adaptive Bitrate 上で用いた Adaptive Bitrate ですが、常に ON にすべきです。 ONにすると、デバイスに繋がっている帯域が自分で設定したビットレートよりも小さくなった場合に自動的に適応してくれます。

■オーディオ マイクをONにするには、画面下端から上へスワイプしてオーディオメニューを開きます。スピーカーのアイコンをクリックして、マイクのミュートを切り、ONにします。映像にマイク音が載ります。

注意。iOSデバイスのボリュームは0まで下げましょう。0以外にしてしまうと、（LiveAirSoloのマイクをONにしている場合）音声がループしてしまいます。というより寧ろ、音声がループしていたらココを疑って下さい。

■カメラ 上記オーディオメニューを開くと、右に在る歯車を押せば、カメラメニューが現れます。

■チャットを載せる Settings（画面上端をスワイプして現れる青い歯車）から、Extra Features をクリックして Twitch Chat を 360円 でお買い求めください。 入手されましたら、まずは Go Live を押して開始してのち、上のメニューから チャットアイコンをクリックする順番でないと動作しません。

■タイトル/カテゴリ 現状、Twitch チャンネルのご自身の タイトル/カテゴリ を変更する方法はこのアプリから見つかっていません。ブラウザのTwitchページから変更頂くしか無いです。

■Bluetoothのススメ 高性能の Bluetooth イヤホン/ヘッドホン をマイクとして用いることで、スマホのカメラを好き放題動かしても、音声が常にしっかり入ります。スマホから配信なさる場合は意外と便利です。
