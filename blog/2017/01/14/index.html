2017年1月 世界WiiUスマブラーTop16情報

「スマブラ」総合大会 “Genesis4” が今年もやってきます。毎年恒例の「年始トッププレイヤー紹介」です。2017年1月20-22日に開催されるこの大会の、Top16シードを、プロシーンの側面から紹介します。

#Genesis とは

Genesis は「スーパーメジャー」大会です。スマブラは基本有志での開催なので大会数が非常に多く、アメリカはいざしらず、日本だけでも2016年 21都道府県で125回も大会が オフラインで開催。そのためコミュニティが呼ぶ大会格付けが有り、Local, Regional, National, Major と並び、EVO, Genesis ともなると SuperMajor となる。いずれも通称。（日本最大の大会は National に位置づけられ、この前の 2GG ZeRo Saga 等は Major と呼ばれている。）基本的に、偉い人や偉い機関など権威が格を決める訳でもなく、「スマブラーがどれだけ気にしたか」が何となくの指標となる。
そんな中でGenesis4は世界の10カ国以上からプレイヤーが集まり、日本からも（WiiU+DXで分かっている限り）30名以上が参加。略称は #G4 。

  • Smashgg ページ：https://smash.gg/tournament/genesis-4/details

（生放送情報は調査中です）

昨年、Genesis, TheBigHouse, Apex, EVO といった大会は Nintendo of America のオフィシャルサポートがありました。

#Top16とは？ {% oembed https://twitter.com/BearUNLV/status/817552496029614081 %}
（ツイートが消えたときのためにこちらに <a href="/images/Genesis4_01.jpg>画像データ）

そのGenesis4（#G4）のTop16シードが以上のメンバーである。（トーナメント運営のBear氏、お疲れ様です。）誰が誰なのか、見ていきましょう。
（スマブラ世界ランキングは、WiiU を Panda Global が・DX を MIOM が出していますが、PandaGlobal（以下PG）はその最新版が未発表ゆえ、Genesis4のシードを今回紹介。）
（引用したツイートは特に意味はなく、本人の像が分かるようなモノを引っ張っています。）

1. ZeRo

{% oembed https://twitter.com/TSMZeRo/status/818577115067150337 %}
ロサンゼルスesportsアリーナ。Genesis前にファンミーティングを開催する男。
ZeRo。チリ出身。2016年PG世界ランキング1位。私の旅日記では「絶対王者」を号している。前回のGenesis3タイトルホルダー。EVO2連覇。スマブラ大会の連覇記録でギネスを持つ。彼にとってスマブラの偉業は人生のクリアゲッターに過ぎない。来日経験2度。
所属はTSM（TeamSoloMid）。私が勝手に北米LoLの「御三家」と呼んでいるチーム。チームメイトはDX界のグラジオ：Leffen。

2. MKLeo

{% oembed https://twitter.com/Mkleosb/status/812098534048571392 %}
メキシコ出身。まさかの21世紀生まれで非常に若い。2016年初めてUSAの地を踏み、直近の 2GG ZeRo Saga をマルス・クラウドを以て制覇。ZeRoおよび世界の強豪を全て抑えての優勝であった。これを受けて#G4第2シードと破格の対応。
所属は Echo Fox（えこーふぉっくす）。オーナーは元NBA選手 Rick Fox。故にチームがプライベートジェットまで所有。LoLは北米ChallengerSeries 参加、CoD, CS:GOも強豪。チームメイトにはお馴染みのMew2King、及び格ゲーの チョコももち・ときど といった面々が連なる。

3. Larry Lurr

{% oembed https://twitter.com/LarryLurr/status/819937077483683840 %}
アメリカ西海岸南：Socal元祖の古豪。スマブラX のファルコ時代は DEHF（Does Everybody Hate Falco ?）と名告っていた。
1月Misfits所属となった。（Misfitsは由来が複雑でネットでの我が調査は難航した。元祖 Misfits はのちに Renegades へ改名した。その Renegades の姉妹チームとして派生した北欧チーム Renegades:Banditos が再ブランドとして改名し、Misfits となった。誤りがあればツイッターで指摘願いたい。）本チームはLoLでヨーロッパ Challenger Series 参加。Overwatch, HoTS でも活躍。チームメイトはDXの The Moon。
その前は eLevates に所属。

4. Ally

{% oembed https://twitter.com/C9AlIy/status/816310134863523840 %}
みんな大好き、カナダのマリオ使い。EVO2016王者。ウメブラFATで来日。日本好きで、FFは9派。
所属は Cloud9。上述LoL「御三家」の一つ。チームメイトにDXのMango。

5. VoiD

{% oembed https://twitter.com/gsmVoiD/status/819389280334155778 %} ハワイ出身、現在Socalに住まうシーク使い。WiiU版からの参戦。ウメブラ25で来日。少々日本語を使う。
所属は CLG（CounterLogicGaming）。上述「御三家」最後の一つ。チームメイトは NAKAT・DXの SFAT, PewPewU。CLGの社員に Keitaro が居る。

6. Nairo

{% oembed https://twitter.com/NairoMK/status/819756076061036544 %}
96年生まれの若いプレイヤー。スマブラXの頃から活躍したスター。Nairoの生放送は大人気。既に3度来日。FFは10派。
所属は NRG。NRGはLoLのChampionship Seriesに参加。CS:GO, Overwatch, Hearthstone 等、多彩なゲームにプレイヤーを持つ。出自は、NBAのチーム Sacramento Kings が TeamCoast のLoL部門を買収して成立した所から。後にMLBのアレックス・ロドリゲスといったスター達も出資を発表。
Nairoは以前 Team Liquid Pro に所属していたが、NRGへの移籍に際して “Because I can grow together” と表現。栄えある新星チームで、旗振り役を任されている。

7. komorikiri

{% oembed https://twitter.com/komorikiri_SSCF/status/798685484431966208 %} 日本の関西を拠点とする：古森霧 選手。日本プレイヤーランキング1位。（Smash Records : Japan Power Rankings）昨年のGenesis3にてダブルスは優勝。その際のペアは らない 選手。後述。
2GG所属。2GGはイベント運営を主力とする。Los Angeles esports Arena を拠点に 2GG Championship Series の開催を発表。合計賞金1000万円+の年間定期シリーズ。柿落としの 2GG Genesis Saga はカップヌードルがスポンサーに着いていた。

8. ANTi

{% oembed https://twitter.com/Anti/status/819791972235739136 %} ANTi。ギャグキャラとして他ゲーファンであろうとも名は音に聞く。「Apex2012本戦寝坊」といったサボり伝説は数知れず。「週40時間も仕事してたらプロにはなれない」など格言は後を絶たない。多キャラ使い。CEO2016 優勝。
Immortals 所属。LoLは北米Championship Series参加。CS:GO, Overwatch チームも所持。チームメイトにDXのShroomed。

9. Abadango

{% oembed https://twitter.com/LG_Abadango/status/784871626215227403 %} 日本：関東のあばだんご。2016年PG世界ランキング5位。日本SmashRecordsランキング13位。
北米の Luminosity Gaming 所属。チームメイトにDXのIce。日本にプロゲーマーは多くなったが、実は「海外のチームに所属」というプレイヤーは少ない。
経緯にあたっては、詳しくはこちらをどうぞ。

10. Dabuz

{% oembed https://twitter.com/DabuzSenpai/status/820494870124998656 %} ニューヨークを拠点とするプレイヤー。安定感が高いロゼッタ使い。昨年のGenesis3で準優勝。ウメブラSATで来日。
所属はRenegades。3.Larryでも述べた、由来が複雑なチーム。とりあえずMisfitsとの姉妹チーム、拠点はデトロイトとおぼえましょう。CS:GO, Haloが強い。チームメイトにDXの Swedish Delight。

11. Mr.R

{% oembed https://twitter.com/Mr_RSmash/status/820498936414801921 %} オランダのプレイヤー。シーク使い。ウメブラSATで来日。
スマブラ界上位勢に残された数少ない Free Agent（無所属）。しかしオランダのビザは難しいため、手をこまねいて看るチームが多い。グローバルゲームのプロ化は様々な事情が有る（DXではスウェーデンのLeffenが殊に）。

12. Kameme

{% oembed https://twitter.com/kameme8808/status/820284508230578176 %} かめむし/Kamemushiと同一人物。日本：関東のプレイヤー。初遠征であるEVO2016で準優勝したロックマン。最近はシークも用いる。
日本の DetonatioN Gaming 所属。チームメイトに にえとの選手も。マネージャーが Rain 選手。

13. Ranai

{% oembed https://twitter.com/hysranai/status/813174946121674753 %} らない 選手。日本：関西のプレイヤー。通称「村長」。昨年のGenesis3にて3位。EVO2016ベスト6。レアキャラである むらびと 使い。昨年は帰国前にバーガー屋で店員にサインを求められた。
上のツイートにあるが、今回 2GG の支援にて#G4参戦が決定。

14. Marss

{% oembed https://twitter.com/Marss_NE/status/805078995213676544 %}
ロードアイランドに住むゼロスーツサムス使い。98年生まれの満18歳。昨年のGenesis3にて、ダブルスで ZeRo&Nairo を下し名を轟かせる。シングルスでベスト8入りを果たし、一挙に列強入りを果たした。兄が Pugwest。
所属は Denial eSports。チームは “Wolf Pack” の異名でも呼ばれる。複数ゲームタイトルを保持するチームだが、CoDで2015世界制覇が著名。

15. Salem

{% oembed https://twitter.com/MVG_Salem/status/772530520777523200 %} 昨今浮上して来たSalem。東海岸のベヨネッタ使い。以前はスマブラXにてApex2013年を征した戴冠者。当時ダークホース過ぎて Ally をして「いいか、Salem が優勝したら、俺はスマブラXを辞める。約束する。」とまで言わしめたプレイヤー。（結局優勝したが、Allyは引退しなかった。）
所属は MVG。大会運営を主としており、MVGは5月にウメブラとのコラボのために来日予定（Salemは？）。Salem は Clash Tournaments とも依然関係がある。

16. Tweek

{% oembed https://twitter.com/TweekSsb/status/820192979209990144 %} 東海岸のクラウド使い。以前はクッパJr.も用いていた。98年生まれの18歳。PG世界ランキング15位。直近の 2GGC Genesis Saga にて4位であり、他のシードプレイヤーにとって全く油断ならないだろう。
所属なしの Free Agent。

#以上 ここまで16名のシード選手が並んだが、これは強豪の全てではない。NAKAT, False, ESAM, Pugwest, Mr.E, Rich Brown, 6WX, Samsora などなど。キリがない。日本から行くメンバーも多角多彩（誰かまとめてください…）。
しかし、これら16シードを基に「チーム」「出身地」といった見地から、他ゲーも絡めて、大会を楽しむ手法も知って頂ければ幸いである。

次は、今年もGenesis旅日記を書く予定なので、またその際にお会いしましょう。
