プリキュアを始める方法 2019

「プリキュア気になるけれど、どう見始めたら良いか分からない」という方へ。プリキュア開始から満15年経った現在、何から取り掛かるのがオススメか記していこうと思います。

導入

『プリキュア』シリーズは、テレ朝系列のみなさん・東映アニメーションさん・ADKさんによって制作されるシリーズで、なんと2004年2月に開始し、毎年続けているアニメ作品です。
（ほかの2004年開始アニメ作品：魔法少女リリカルなのは・BLEACH・ローゼンメイデン・機動戦士ガンダムSEED Destiny …etc）

作品が所謂「女児向けアニメ」ということで、新規ファンのお子さんが必ず毎年発生します。それに伴い、その保護者の方もプリキュアをご覧になるため、毎年新規視聴者が発生する特殊なアニメです。筆者も周りに「プリキュアを見始めた」という方が多くなってきたのですが、そこで：

  • 「プリキュアで子供と会話することが不可能」
  • 「小児科医なんだけど患者とプリキュアの話題、不可避」
  • 「面白いからもう一歩踏み込んでプリキュアを視たい」

という方に遭遇します。特に最後の方は、踏み込みたいけれどどこから始めれば良いか迷子になっています。そこで、そういった方々にプリキュアの導入への手引きを書こうと思い立ちました。

ただ、プリキュアには様々な側面があり、まとめてに評価することは難しいため、カテゴリ別にプリキュアの楽しみ方を分類しました。まずはお子さんと同じプリキュアをご覧になるのがベストなのですが、更に他の作品をご覧になる方のための、主観的なランキングです。

![“代替メッセージ”](/images/Precure/スクリーンショット 2019-03-19 19.41.21.png)
[ Amazon より。『スター☆トゥインクルプリキュア主題歌シングル』]

まず、プリキュアは毎年別作品（『MaxHeart』および『5GoGo』を除く）であり、世界は繋がっていません。例外的に劇場版「オールスター」に当てはまる作品（など）にて異なる世代が一緒に登場することもあるのですが、基本的には独立した作品から構成されます。

そのため、どこから始めても良いのですが、確かに多すぎます。筆者は15年をかけて全部視ているから可能なのであって、普通は不可能です。稀に「途中から見始めて、地上波オンエアを全部完走した」という方がいらっしゃるのですが、気が狂っています。（プリキュアは2019年3月現在で約750話。）

では順を追って、ご提案申し上げます。

#劇場版

プリキュアを初めてご覧になるのであれば、劇場版が最もオススメです。プリキュアの映画は長くとも70分であり、オールスター作品と単品作品に分かれており、「広く薄く」も「短くしっかり」も双方テイスティング可能です。

プリキュアは1年に2作品の劇場版が制作されます。春と秋に分かれており、春がオールスター・秋が単品という慣例になっています。（例外アリ。）

1. 映画 プリキュアオールスターズ みんなで歌う♪奇跡の魔法!

（世代：魔法つかいプリキュア、オールスター作品, 2016年）


[ Amazon より]

名作。

プリキュアシリーズ長しと雖も、ミュージカル作品はこちらが唯一です。劇中歌の作詞は森雪之丞さんが担当されました。
オールスターの春映画ですので、歴代のプリキュアが概ね登場し共闘します。プリキュアの勝利へのアプローチを描き、その上で個人個人を描写した作品となっています。オールスターの集大成と云っても過言ではないでしょう。プリキュアらしさを体験し、各世代を網羅的にに味わいたい方の導入には最適かと考えます。

2. 映画ハピネスチャージプリキュア！人形の国のバレリーナ

（単品作品, 2014年）


[ Amazon より]

こちらは「ハピネスチャージプリキュア」一代のみが登場する秋映画です。「ハピネスチャージ」と名を冠するだけあり、幸せを追求する作品なのですが、人の幸せは中々叶わないというコンセプトになっています。この映画でも、プリキュアに解決できない現実が突きつけられます。
デザインの面では、普段の地上波版とは異なる専用日常コスチュームが用意されており、全体的な絵の描き込みも細かくなっています。

挿入歌『勇気が生まれる場所』は傑作です。今作の声優である 中島愛・潘めぐみ・北川里奈・戸松遥（敬称略）による名曲です。初めて聴かれる方は「プリキュアってこれ程重い曲があるのかあ」と意外がられるかと思います（すみません、曲の評価はテキトーです）。

ハピネスチャージプリキュアについては、以下の記事が詳しいのでご覧になってください。↓
記事： 「映画ハピネスチャージプリキュア！」の興行収入はなぜ半減した？　2014年の出来事から考える - ねとらぼ

3. 映画ドキドキ！プリキュア マナ結婚!!?未来につなぐ希望のドレス

（単品作品, 2013年）


[ Amazon より]

『ドキドキ！プリキュア』一代のみが登場する秋映画です。下でもドキドキプリキュアについては紹介しますが、劇場版は本編とは少し逸れた内容となっています。「愛」がテーマのプリキュアで、本編では主人公のキュアハートは常に気丈なキャラなのですが、映画では珍しくキュアハートの 絶望・傷つき が描かれます。

筆者の憶えている限り、プリキュアで唯一流血が描かれる作品です。

その他

他にも、オールスターをしっかり楽しみたい方は『映画 プリキュアオールスターズNewStage』（2012 - 2014年）シリーズが3部作になっておりますので、3作続けてご覧になることをオススメします。
また、『映画 ハートキャッチプリキュア! 花の都でファッションショー…ですか!?』（2010年）が惜しくも自分の中で4位だったのでここで言及します。パリの描写はかなり凝っています。

いずれも Netflix の方ならばそのまま視聴可能ですが、Amazon Prime 会員の方も有料ストリーミングで楽しむことも可能です。

#50話視たい作品

映画から一転し、こちらはじっくり腰を据えてご覧になりたい方のためのカタログです。プリキュア、アニメ本編は4クールの約50話が基本ですから、気合を入れねばなりません。その分、

  • 50話をかけて見る意味がある作品
  • 1年をかけて制作をする意味がある作品

を撰抜したつもりです。

ちなみにですが、“初代” の『ふたりはプリキュア』だけをご覧になった方は「初代はすごかった」「初代以外をプリキュアと認めない」といった意見をおっしゃるのですが、全プリキュアを視聴して分類しようと考えると、寧ろ「初代が例外的」という感想を抱きます。実はプリキュアには、それ以降素晴らしい作品が山程あるのです。

1. Go!プリンセスプリキュア

（2015年）


[ Amazon より。こちらは画集です。]

最高傑作。

テーマは一言でまとめると「夢」です。主人公：はるかは庶民的な方ですが「プリンセスになりたい」という曖昧な夢を持ってお嬢様学校に入ります。実態がなく・抽象的である夢を持った主人公が、周りのお嬢様と接したり、悪の帝国の “姫”（文字通り）と戦ったりする中で、強い結論を導く1年です。タイトルからして、

  • 「プリンセスってなんか軟弱そう」
  • 「肉体派のプリキュアらしくない」

と毛嫌いする浅はかな方もいらっしゃるのですが、寧ろ内容はその逆です。

プリキュアは初代の時点で「ヒーローの男が助けに来ない」という概念が斬新でした。プリキュアの開始は2004年で、『美少女戦士セーラームーン』の一通りのオンエア終了からたった7年後ですから、前進的であったことがうかがえるでしょう。『プリンセスプリキュア』では、その「ヒーローの要らない女性ヒロイン像」を更に脱ぎ捨て、次のステージを提案する内容となっています。一人の少女が夢に目覚めてから、「切磋琢磨して自分を磨く」「男性ヒーローを捨て去る」「プリンセスを定義付ける」といったステップを丁寧に描きます。

![“代替メッセージ”](/images/Precure/スクリーンショット 2019-03-25 17.23.19.png)
[ 東映アニメーションミュージアム公式YouTubeチャンネル より。第一話。グラデーションのかかった髪から、拳が飛び出します。]

東映アニメーションさんの公式サイト に各話あらすじがあり、こちらを読むだけでも一通りの理解は出来るかと思います。

2. ドキドキ!プリキュア

（2013年）

「愛」を追究した一作。


[ Amazon より。]

おそらく登場人物の人間性・戦闘力ではプリキュア史上最強のメンバーが揃った作品。その分、テーマの「愛」が重く、敵も強大です。 過去のブログ でも記しました。身近な人を想う愛情と、広く万民を想う愛情は、相反するもの。この2つが武力として激突する作品です。

決して折れることのない主人公から、爽快感と安心感が伝わってくる、実にプリキュアらしい一作です。

3. ハートキャッチプリキュア！

（2010年）


[ Amazon より。]

テーマは「チェンジ」（当時オバマ大統領が就任したため）。コンプレックスを抱いた人物たちが、自分を変えるという作品です。花が全体のアートモチーフになっています。

コンプレックスに焦点が当たりますので、全体的に重めの内容となっています。自分の弱点との向き合い方は様々であり、努力して能力を伸ばす人も・弱さを受け入れる人もいます。人の死がハッキリ描かれる珍しいプリキュアです。

（おまけ：知名度的には “初代” にも引けを取らないでしょう。リアルタイムに視聴していた側としては、それまでプリキュアを見ていると「変態」の烙印を押されていましたが、「ハートキャッチ」で一度社会への露出が広がり、理解と市民権を得られてきた1年だったと感じています。）

■ プリキュアはいずれの作品も、最後は具体的な「人物」の 倒す・倒される の話で終わらないところが魅力です。どういうことかと云いますと、最後は必ず抽象的な「概念」のぶつかり合いになります。
ネタバレを（少し）防止するために他作品を例に出しますと、スイートならば「音楽と雑音」、ハピネスチャージならば「幸せと失恋」。プリキュア陣営と敵陣営が、それぞれ片方を象徴し、概念同士での衝突を起こします。毎回プリキュアは敵を倒すというよりは、なにか新たな次元へ昇華させて結論を提案する結末を迎えます。考えさせられる面白みがあります。

#ダンス

視聴の側面から少し外れて、ダンスについて述べます。

プリキュアのダンスは非常に重要な要素です。昔、子供向けアニメはエンディング映像は静止画またはそれに近い演出が一般的でした。しかし『フレッシュプリキュア』にてCGアニメーションによるモーションキャプチャが導入され、児童向けアニメに革命を起こしました[要出典]。今では児童向けアニメでダンスを活用するのは常套手段です。

おそらく、興味を持ってダンスを楽しみたい大人の方もいらっしゃるでしょう。そんな方へ、以下の作品をオススメします。

1. シュビドゥビ☆スイーツタイム

【キラキラ☆プリキュアアラモード】後期エンディング（2017年）

{% oembed https://www.youtube.com/watch?v=uFfsTeExwbQ %}

最も難易度が高く、おそらく子供に踊らせる気は無かったと考えられる振付。EDムービーも、CGやエフェクトの小細工を余り使わず、ダンスで勝負しています。これらの理由からダンス #1 評価としました。

踊る場合、振付が歌詞ではなく拍に合っているため、前提としてアップで8拍を取る知識が必要です。プリキュアアラモードは前後期共にダンスが難しく、筆者が実際に池袋サンシャインできぐるみショーを拝見したところ、きぐるみの皆さんは完璧には再現していませんでした。

2. イェイ！イェイ！イェイ！

【スマイルプリキュア！】前期エンディング（2012年）

東日本大震災の翌年、日本を元気にすることを目的に「笑顔」をテーマに作られた『スマイルプリキュア』。このダンスはその理念を体現した芸術です。踊りやすく、躍動感のある振付で、踊っているうちに自然と笑顔になります。更に、腕を伸ばしている時間が長く、実際に自分が動くと気持ちが良いです。おそらくウィンクを取り入れた初のEDダンスです。サビだけでも憶えると楽しいです。

「スマイルプリキュア」での前後期EDダンスは、振付師である故：前田健さんの遺した傑作であると筆者は受け止めています。

3. ラブリンク

【ドキドキ！プリキュア】後期エンディング（2013年）

（こちらのダンスは公式映像が見つからなかったため、上記地上波50話か劇場版をご覧になってください。）

上でも紹介した『ドキドキ！プリキュア』の愛を押し込んだダンス。小文字の「e」を体で作るところが実にクリエイティブ。難易度が低めで、楽しくなれます。

ちなみに曲のフル版では、Cメロが有ります。プリキュアOP・ED曲にCメロが入った初めての例です。

その他

難易度で云うと、ハートキャッチプリキュアのED曲『ハートキャッチ☆パラダイス』も簡単めで踊りやすいです。ダンス未経験の方はこちらか、上記『イェイ！イェイ！イェイ！』から取り掛かるのがベストでしょう。

こちらでは例を挙げてオススメを申し上げましたが、ご自身でED映像をご覧になって合うものを探して頂くのがベストです。通称「3Dシアター」や「映画 オールスターズDXシリーズ」らも、ダンスの難易度が初心者には法外で手応えがあります。

#直近のプリキュア

最後に、直近のプリキュアをご紹介します。こちらは1年しか有効でないのですが、今このブログを御覧になっている方には死活問題のはずです。お子さんがいらっしゃる場合、直近2年のプリキュアが必要知識となるからです。

以下の作品の雰囲気を掴むのであれば、2017年-2018年の「秋」の映画作品をご覧になることがオススメです。お子さんと一緒にご家庭で上映するのも風流でしょう。それぞれこちらです：

  • 『映画HUGっと！プリキュア・ふたりはプリキュア オールスターズメモリーズ』
  • 『映画キラキラ プリキュアアラモード パリッと！想い出のミルフィーユ！ 』

HUGっと！プリキュア（2018年）

応援するプリキュアです。

![“代替メッセージ”](/images/Precure/スクリーンショット 2019-03-26 2.51.32.png)
[ 東映アニメーション公式サイト ]

メンバーの名前は（個人的に最も）覚えづらいです。それぞれキュアを頭につけて、「エール・アンジュ・エトワール・アムール・マシェリ」の5名です。特にアムールとマシェリは双方「愛」という意味のため、私も慣れるまで時間がかかりました。残念ながら、この世代の名前は必死で暗記してください。劇場版を見ても名前を憶えることまでは期待できません。

キラキラ☆プリキュアアラモード（2017年）

スイーツを作るプリキュアです。

![“代替メッセージ”](/images/Precure/スクリーンショット 2019-03-26 2.51.57.png)
[ 東映アニメーション公式サイト ]

プリキュアの枕詞は通常「伝説の戦士」ですが、この年は例外的に「伝説のパティシエ」となっています（16作品中、例外はあと1作品です）。スイーツをアートモチーフに用い、全体的にフランス語を散りばめたお洒落なデザインです。

メンバーの名前については、上にも記したED曲『シュビドゥビ☆スイーツタイム』を聴けば憶えられます。「ホイップ・ジェラート・カスタード、マカロン・ショコラにパルフェ」です。リズムにのってすぐ頭に入ります。

この年はまた例外的に、戦闘で拳を使わないため、ご覧になると物足りなく感じることもありますが、ストーリーが進むにつれ内容はシリアスで重い空気になって行きます。「料理には愛が一番大事」を1年かけて体現した独特の作品です。究極的な結論については 過去のブログ でも考察を入れました。

#以上、

筆者は「プリキュアはワインのよう」と申し上げています。様々な銘柄があり、相手に応じてオススメする作品を選択せねばなりません。また、同じビンテージでも年月を置くことによって評価が上がることもあります。

プリキュアは作品も多いので、もし質問がある場合はネットで調べるよりも、ぜひお近くのプリキュアソムリエにおたずねください。
