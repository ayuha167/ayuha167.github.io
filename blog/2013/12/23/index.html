##環境 PC: MacBook Pro, Core i7, Mem 8GB, not Retina
OS: OS X 10.7.3
XCode: 4.6.2

##ちなみに、備忘録 $ rake new_post[title] で記事が作成され、 source/_post/2013-XX-XX-1.markdown というディレクトリに記事のテンプレートが現れる。これを編集してページ作るのです。

Markdown記法に関しては、
kojika17さんのページ で。

簡単に言うと、

  • スペース2個で改行
  • HTML の <えいち1> が “シャープ”, <えいち2> が “シャープシャープ”

です。

あとは $ rake generate で。
プレビュー見るなら $ rake preview で、http://localhost:4000/ にアクセスすれば見られる。

ここでOKなら、 $ rake deploy で公開。
