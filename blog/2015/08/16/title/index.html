スマブラWiiU 2015 最強決定戦 : 第1章 ウメブラF.A.T.を創る人間たち

スマブラって、どんな人がなにを目指してプレイしているの？それをノンスマブラーにも伝えるシリーズ個人ブログです。
この “第1章” は、大会を創る側の記述です。

(第0章)http://ayuha167.github.io/blog/2015/08/15/title/

スマブラ界がいかに盛り上がっているかは、前記事で注目しました。では、実際どのように大会は生きているのか、見て参ります。

0. “ウメブラ” とは ??

スマブラの有志大会のうち、関東エリアを司る “ウメブラ” は、1ヶ月毎に開催されます。この特大版が、今回テーマの “ウメブラ F.A.T.” (= First Anniversary Tournament) です。 この2015年、9月シルバーウィークに東京で開催されます。
有志だけで集まり、ただ「自分たちの中で最強って誰なの？」という題目に絶対の答えを打つためのイベントです。海外勢の参戦も予定されています。

（告知動画＠3月） {% oembed https://www.youtube.com/watch?v=4AP9yltFPfE %}

（大会ツイッター） {% oembed https://twitter.com/UMBRHP/status/630884799113072640 %}

ただのエンジョイイベントには終わらせたくないため、運営も本気です。プレイヤー今までのウメブラと、関西の “スマバト”, マルチ格闘ゲーム大会 “KSB 2015” の結果を合わせてシードポイントを計算し、プレイヤーを順位付けしています。

(シード表) http://piosuma.blog.jp/archives/24157620.html

1. 運営スタッフは語る


(EVO日本語放送。TOPANGAチャンネル http://www.twitch.tv/topangatv/ から。左 : スタッフのエル氏。)

スタッフのエル氏は、ウメブラF.A.T. にあたり、こう述べています :

「自分を含めたスタッフ5名が、EVO2015に行き、本場の盛り上がりを経験しました。自分は初めての海外大会だったので、この業界のポテンシャルを確かめられました。
日本でも、同じことが出来るはずです。それを、F.A.T. で実現したいと思っています。」

2. 大会はかくありけれ

ウメブラスタッフの朝は早い。

日常生活では有り得ない時間(6時台)に起床し、会場へ到着。48台の協賛モニタを箱から出し、ひとつひとつ丁寧に組み立てて行きます。既に朝の時点で、20人ほどのスタッフが勢揃いしています。運搬は主催 : うめき氏の車です。参加者が持ち込んだ同じく48のWiiUをセット。プロジェクタ・配信台も当日朝セットします。


いざ大会が始まればこれもまた大変です。190人エントリーのトーナメントを1日で消化する恐ろしい運営です。予選は全員参加。この190人はちゃんとブロックに割り振られ、各予選テーブルに各々移動してもらっています。16ブロックの総当り。ここは参加者全員が協力して、予選毎に結果記入していきます。

{% oembed https://twitter.com/chocoblanka/status/629864104283340801 %}

ウル4のEvilGeniuses チョコさんが一度お越しでしたが、その際のツイッターです。いい写真ですねえ…。

スマブラの大会は基本オープントーナメントで、参加資格はありません。逆に、選手控室なども無いため、皆が皆と喋れます。強くて有名なあの人も、そこら辺のイスに座っています。予選は午前いっぱいで終わるため、プロも一般も、ごちゃまぜで昼ごはんに行きます。

予選が終わると、本戦トーナメントです。96名ダブルエリミネーション。本戦へ進出出来なかったプレイヤーも、 “Bクラス” というシングルエリミのトーナメントへ出場できます。負けが混んでも、まだまだ大会気分を味わえるのです。これら2トーナメントを、パソコン2台でchallongeを用いて進行します。


（大会本戦前：エル氏が進行の説明をしています。）

エル氏のコメントです：「ウメブラでは、毎回朝一で “初来場の方” に挙手してもらうのですが、それがいつも2割とかいらっしゃいます。そういう方や、プロゲーマーを含めた強豪プレイヤーが同じルールで勝負をすることになるため、しっかりとしたルールの共有が大事ですねえ。実際にルールに依るトラブルは全然起きないため、プレイヤー間での意思疎通は順調に行っていると思います。」
